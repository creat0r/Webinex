A MODX Revolution Extra for managing webinars. 

Features:

- Custom resource class with custom UI for webinars, allowing for multiple presentations
- Custom manager page for managing presenters, companies, videos, documents, and affiliates
- Extended user class for tracking prospects and affiliate referrals
- Snippets for listing upcoming and recorded webinars, presenters, and presentations
- Snippet for tracking affiliate referrals
- FormIt hooks to handle registration and sumbission to an external system
- overThruster plugin to enhance performance by pre-processing templates
